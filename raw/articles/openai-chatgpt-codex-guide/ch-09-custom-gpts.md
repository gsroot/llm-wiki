## Chapter 09. Custom GPTs — 나만의 AI 어시스턴트 만들기  
  
ChatGPT Plus나 Team, Enterprise 요금제를 사용한다면 **Custom GPTs** 기능을 통해 특정 목적에 완벽하게 맞춤화된 나만의 AI 어시스턴트를 만들 수 있습니다. 단순히 프롬프트를 반복 입력하는 것을 넘어, 역할·지식·도구·외부 연동까지 모두 사전 설정된 전용 AI를 제작하고 공유하는 것이 Custom GPTs의 핵심입니다.

## 

## Custom GPTs란 무엇인가?

**Custom GPTs**는 특정 용도에 맞게 사전 구성된 ChatGPT의 커스텀 버전입니다. 예를 들어 "우리 회사 HR 정책 전문가", "파이썬 코드 리뷰어", "영어 이메일 교정 도우미"처럼 목적이 명확한 전문 AI를 만들 수 있습니다.

일반 ChatGPT와의 차이점을 비교하면 다음과 같습니다.

항목 | 일반 ChatGPT | Custom GPT  
---|---|---  
역할 설정 | 매번 프롬프트로 지정 | 사전 저장, 즉시 적용  
전문 지식 | 학습 데이터에 한정 | 파일 업로드로 추가 가능  
외부 연동 | 불가 | API Actions로 가능  
  
Custom GPT는 **GPT Store** 를 통해 다른 사람이 만든 GPT를 가져다 쓸 수도 있고, 직접 만든 GPT를 팀 또는 공개적으로 배포할 수도 있습니다.

## 

## GPT Builder 사용법

Custom GPT를 만들려면 ChatGPT 좌측 사이드바에서 **"Explore GPTs"** → **"+ Create"** 버튼을 클릭합니다. 빌더 화면은 두 가지 탭으로 구성됩니다.

**Create 탭 (대화형 빌더)**

GPT Builder와 대화하면서 자동으로 설정이 완성되는 방식입니다. "어떤 GPT를 만들고 싶으세요?"라는 질문에 자연어로 답하면, Builder가 이름·설명·지침을 자동으로 생성해 제안합니다. 초보자에게 권장하는 방식입니다.
    
    
    예시 대화:
    나: 우리 회사 복리후생과 사내 규정에 대해 답변해주는 HR 도우미 GPT를 만들고 싶어요.
    Builder: 알겠습니다. "HR 도우미"라는 이름은 어떤가요? 직원들의 복리후생, 휴가 정책, 사내 규정에 관한 질문에 친절하게 답변하는 GPT로 설정하겠습니다.
    

**Configure 탭 (직접 설정)**

세부 항목을 직접 입력하여 정밀하게 제어하는 방식입니다. 실무에서는 Configure 탭을 주로 사용합니다. 각 구성 요소는 다음 섹션에서 자세히 설명합니다.

## 

## 구성 요소 상세 설명

### Name과 Description

  * **Name** : GPT의 이름입니다. 간결하고 용도가 명확한 이름을 권장합니다.
  * 좋은 예: "HR 도우미", "마케팅 카피라이터", "파이썬 코드 리뷰어"
  * 나쁜 예: "My GPT 1", "테스트용"

  * **Description** : GPT Store나 목록에 표시되는 한 줄 설명입니다. 어떤 상황에서 이 GPT를 쓰면 좋은지 명확히 적습니다.


### Instructions (시스템 프롬프트)

**Instructions**는 GPT의 핵심 행동 지침입니다. 일반 프롬프트와 달리 항상 백그라운드에서 적용됩니다. 효과적인 Instructions 작성 원칙은 다음과 같습니다.
    
    
    [Instructions 예시 — HR 도우미]
    
    당신은 (주)ABC 회사의 HR 도우미입니다.
    
    역할:
    - 직원들의 복리후생, 휴가, 급여, 사내 규정 관련 질문에 답변합니다.
    - 업로드된 사내 문서를 참고하여 정확한 정보를 제공합니다.
    - 문서에 없는 내용은 추측하지 말고 "HR팀에 직접 문의하세요"라고 안내합니다.
    
    응답 스타일:
    - 친절하고 간결하게 답변합니다.
    - 관련 문서의 페이지나 섹션을 인용합니다.
    - 항상 한국어로 답변합니다.
    
    금지 사항:
    - 개인 급여나 인사 평가 정보는 제공하지 않습니다.
    - 법적 조언으로 해석될 수 있는 내용은 피합니다.
    

### Conversation Starters

사용자가 GPT를 처음 열었을 때 보이는 추천 질문 버튼입니다. 자주 묻는 질문이나 GPT를 어떻게 쓰면 좋은지 안내하는 역할을 합니다.
    
    
    예시 Conversation Starters:
    - "연차 휴가는 어떻게 신청하나요?"
    - "육아휴직 기간과 절차를 알려주세요"
    - "경조사 지원금 기준이 어떻게 되나요?"
    - "재택근무 신청 방법을 알려주세요"
    

### Knowledge 파일 업로드

**Knowledge** 기능을 사용하면 PDF, Word, TXT 등 사내 문서를 GPT에 직접 업로드할 수 있습니다. GPT는 사용자 질문에 답할 때 이 파일들을 검색(RAG)하여 답변의 근거로 활용합니다.

지원 파일 형식 | 최대 파일 크기 | 최대 파일 수  
---|---|---  
PDF, DOCX, TXT, CSV 등 | 파일당 512MB | 20개  
  
tip

Knowledge 파일은 GPT가 직접 인용하거나 참고합니다. 민감한 정보(개인정보, 기밀 영업 정보 등)가 포함된 문서는 업로드 전에 반드시 검토하세요. 공개 GPT로 배포 시 파일 내용이 외부에 노출될 위험이 있습니다.

### Capabilities (기능 활성화)

GPT에서 사용할 추가 기능을 선택합니다.

기능 | 설명  
---|---  
Web Browsing | 인터넷 검색으로 최신 정보 조회  
DALL-E Image Generation | 이미지 생성 기능 활성화  
Code Interpreter | 파일 분석, 데이터 처리, 차트 생성  
  
### Actions (외부 API 연동)

**Actions**는 Custom GPT에서 가장 강력한 기능입니다. OpenAPI 스키마를 정의하면 GPT가 외부 서비스의 API를 직접 호출할 수 있습니다. 예를 들어 날씨 API, 사내 인사 시스템, 프로젝트 관리 도구 등과 연동할 수 있습니다.
    
    
    {
      "openapi": "3.1.0",
      "info": {
        "title": "사내 휴가 시스템 API",
        "version": "1.0.0"
      },
      "paths": {
        "/leave/balance": {
          "get": {
            "summary": "잔여 연차 조회",
            "operationId": "getLeaveBalance",
            "parameters": [
              {
                "name": "employee_id",
                "in": "query",
                "required": true,
                "schema": { "type": "string" }
              }
            ]
          }
        }
      }
    }
    

## 

## GPT Store 활용하기

직접 만들지 않아도 다른 사람이 만든 유용한 GPT를 바로 사용할 수 있습니다. ChatGPT 좌측 사이드바의 **"Explore GPTs"** 에서 카테고리별로 탐색하거나 검색할 수 있습니다.

주요 카테고리로는 글쓰기(Writing), 생산성(Productivity), 리서치(Research & Analysis), 교육(Education), 프로그래밍(Programming) 등이 있습니다. 검색창에 "이력서", "코드 리뷰", "번역" 등 원하는 기능을 입력하면 관련 GPT를 찾을 수 있습니다.

tip

GPT Store에서 GPT를 선택할 때는 **사용 횟수** 와 **리뷰** 를 함께 확인하세요. OpenAI 공식 배지가 있는 GPT는 검증된 제작자가 만든 것입니다.

## 

## 공유 설정

만든 GPT를 어떻게 공개할지 설정할 수 있습니다.

공유 옵션 | 대상 | 활용 상황  
---|---|---  
Only me | 본인만 | 개인 업무용 비공개 GPT  
Anyone with a link | 링크 보유자 | 팀 내부 배포  
Public (GPT Store) | 전체 공개 | 외부 공유·GPT Store 등록  
  
## 

## 실습: 사내 FAQ 답변 봇 만들기

### 실습 시나리오

**목표** : 회사 내규와 복리후생 문서를 기반으로 직원 질문에 자동 답변하는 HR 도우미 GPT 만들기

**준비물**

  * ChatGPT Plus 이상 계정
  * 사내 복리후생 안내서 PDF (없다면 샘플 텍스트 파일 사용)


### 단계 1: GPT Builder 열기

ChatGPT 좌측 사이드바 하단의 **"Explore GPTs"** → **"+ Create"** 클릭

### 단계 2: Configure 탭에서 기본 정보 입력

**Name** : `HR 도우미`

**Description** : `사내 복리후생, 휴가, 사내 규정에 관한 질문에 즉시 답변합니다.`

### 단계 3: Instructions 입력

아래 내용을 Instructions 란에 붙여넣기합니다.
    
    
    당신은 (주)ABC 회사의 친절한 HR 도우미입니다.
    
    [역할]
    - 업로드된 사내 문서를 기반으로 복리후생, 휴가, 급여, 사내 규정 관련 질문에 답변합니다.
    - 문서에 명확한 내용이 있을 경우 해당 내용을 인용하여 답변합니다.
    - 문서에 없는 내용은 "정확한 안내를 위해 HR팀(hr@abc.com)에 직접 문의해주세요."라고 안내합니다.
    
    [응답 형식]
    - 핵심 답변을 먼저 제공합니다.
    - 필요 시 번호 목록으로 단계를 설명합니다.
    - 항상 친절하고 간결한 한국어로 답변합니다.
    
    [주의사항]
    - 개인별 급여나 인사 평가 정보는 다루지 않습니다.
    - 모든 내용은 안내 목적이며, 법적 효력이 없음을 필요 시 명시합니다.
    

### 단계 4: Conversation Starters 추가
    
    
    연차 휴가 신청 방법이 궁금해요
    경조사 지원금 기준을 알려주세요
    재택근무 정책이 어떻게 되나요?
    건강검진 지원은 어떻게 받나요?
    

### 단계 5: Knowledge 파일 업로드

**Knowledge** 섹션의 **"Upload files"** 버튼을 클릭하여 사내 복리후생 안내서 PDF를 업로드합니다. 파일이 없다면 아래 내용을 `hr_policy.txt` 파일로 저장하여 업로드합니다.
    
    
    [ABC회사 복리후생 안내]
    
    연차 휴가:
    - 입사 1년 미만: 월 1일 발생 (최대 11일)
    - 1년 이상: 연 15일, 이후 2년마다 1일씩 추가 (최대 25일)
    - 신청: 그룹웨어 > 인사관리 > 휴가신청
    
    경조사 지원:
    - 본인 결혼: 50만원 + 5일 휴가
    - 직계존속 사망: 30만원 + 5일 휴가
    - 자녀 출생: 20만원 + 3일 휴가
    
    재택근무:
    - 주 2일 재택 가능 (팀장 승인 필요)
    - 신청: 매주 월요일까지 다음 주 일정 제출
    

### 단계 6: 저장 및 테스트

오른쪽 상단 **"Save"** 버튼 클릭 → 공유 범위 선택 (**"Only me"** 또는 **"Anyone with a link"**) → 저장 완료

저장 후 우측 미리보기 창에서 "연차 휴가 신청 방법이 궁금해요"를 클릭하여 정상 동작을 확인합니다.

tip

GPT를 만든 후에도 Configure 탭에서 언제든지 수정할 수 있습니다. 직원들의 피드백을 받아 Instructions를 개선하고 Knowledge 파일을 최신 버전으로 교체하면 GPT 품질이 지속적으로 향상됩니다.
