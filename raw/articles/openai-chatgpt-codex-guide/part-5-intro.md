## 핵심 요약 (자가 작문)

Part 5는 OpenAI가 2025년에 출시한 클라우드 기반 자율 코딩 에이전트 Codex를 4개 챕터에 걸쳐 다룬다. Codex는 IDE 자동완성과 달리 자연어 태스크를 받으면 GitHub 리포지토리를 직접 읽고 쓰며 샌드박스에서 코드를 실행한 뒤 Pull Request까지 만들어 내는 "엔지니어형" 에이전트다. 13장에서 Codex의 정체성과 GitHub Copilot·Claude Code와의 비교, 환경 설정을 다루고, 14장은 코드 작성·리팩토링, 15장은 버그 수정·테스트 자동화, 16장은 레거시 마이그레이션·문서화·보안 스캔 등 실전 패턴으로 확장된다. 입문 → 활용 → 패턴화 흐름으로 개발 워크플로우 전체를 Codex 기반으로 재설계하는 것이 학습 목표다.
