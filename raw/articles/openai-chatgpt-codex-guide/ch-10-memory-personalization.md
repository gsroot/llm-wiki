## Chapter 10. Memory와 개인화 — AI를 나에게 맞추기

매번 ChatGPT와 대화를 시작할 때마다 "저는 마케터예요", "저는 한국어 응답을 선호해요"처럼 반복해서 자신을 소개한 경험이 있으신가요? **Memory**와 **Custom Instructions** 기능을 활용하면 ChatGPT가 나의 직무, 선호, 프로젝트 맥락을 기억하고 모든 대화에 자동으로 적용합니다. AI를 범용 도구에서 나만의 전용 어시스턴트로 전환하는 핵심 기능입니다.

## 

## Memory 기능 소개

ChatGPT의 Memory는 대화 간 정보를 지속적으로 기억하는 기능입니다. 사용자가 어떤 정보를 공유하면 ChatGPT가 이를 메모리에 저장하고, 이후 대화에서 자동으로 활용합니다.

예를 들어 어느 날 "저는 B2B SaaS 스타트업의 마케팅 팀장이에요"라고 말하면, 다음번 대화에서 마케팅 전략을 물어볼 때 ChatGPT는 B2B SaaS 맥락에 맞는 답변을 자동으로 제공합니다.

tip

Memory 기능은 ChatGPT Plus, Team, Enterprise 요금제에서 사용 가능합니다. 무료 플랜에서는 제한적으로 지원될 수 있습니다.

## 

## Memory 활성화 및 관리

### 활성화/비활성화 방법

Memory 기능은 기본적으로 활성화되어 있습니다. 설정 경로는 다음과 같습니다.
    
    
    Settings(설정) → Personalization(개인화) → Memory → 토글 ON/OFF
    

### 저장된 Memory 확인 방법
    
    
    Settings → Personalization → Manage memory
    

여기서 ChatGPT가 자동으로 저장한 항목들을 모두 확인할 수 있습니다. 각 항목 옆의 삭제 버튼(X)으로 특정 기억만 삭제하거나, **"Clear all memories"** 버튼으로 전체 초기화가 가능합니다.

### Memory에 저장되는 정보 유형

ChatGPT는 대화 중 다음과 같은 종류의 정보를 자동으로 기억합니다.

정보 유형 | 예시  
---|---  
직업과 역할 | "B2B SaaS 스타트업 마케팅 팀장"  
선호 언어/형식 | "응답은 항상 한국어로, 글머리 기호 형식 선호"  
진행 중인 프로젝트 | "현재 신제품 런칭 캠페인 준비 중"  
기술 스택 | "주로 Python과 AWS를 사용"  
개인적 선호 | "간결한 답변 선호, 이모지 사용 금지"  
  
직접 기억을 추가하고 싶다면 대화 중 "이걸 기억해줘: 나는 항상 요약본을 먼저 받고 싶어"처럼 명시적으로 요청할 수도 있습니다.

## 

## Custom Instructions 상세 설정

**Custom Instructions**는 Memory보다 더 체계적인 개인화 도구입니다. 두 가지 핵심 섹션으로 구성됩니다.

**접근 경로** : 우측 상단 프로필 아이콘 → **"Customize ChatGPT"** (또는 Settings → Personalization → Custom Instructions)

### 섹션 1: "What would you like ChatGPT to know about you?"

ChatGPT가 나에 대해 알아야 할 배경 정보를 입력합니다. 직업, 전문 분야, 프로젝트 맥락, 사용하는 기술 등을 포함합니다.
    
    
    [마케터 예시]
    - 직업: B2B SaaS 스타트업(직원 50명) 마케팅 팀장
    - 주요 채널: LinkedIn, 이메일 마케팅, 콘텐츠 마케팅
    - 타겟 고객: 중견기업 IT 의사결정자
    - 현재 과제: MQL 증대, CAC 절감
    - 사용 도구: HubSpot, Google Analytics, Figma
    
    
    
    [개발자 예시]
    - 직업: 풀스택 개발자 (경력 5년)
    - 주요 스택: Python(FastAPI), React, PostgreSQL, AWS
    - 현재 프로젝트: SaaS 대시보드 개발
    - 코딩 스타일: 타입 힌트 필수, 함수형 스타일 선호
    - 관심사: 시스템 설계, 성능 최적화
    
    
    
    [연구자 예시]
    - 직업: 의학 연구원 (임상 시험 전문)
    - 연구 분야: 종양학, 신약 개발
    - 사용 언어: 논문은 영어, 일상 업무는 한국어
    - 데이터 분석 도구: R, SPSS
    - 소속: 대학 병원 연구소
    

### 섹션 2: "How would you like ChatGPT to respond?"

ChatGPT의 응답 방식, 형식, 어조를 지정합니다.
    
    
    [공통 추천 설정]
    - 응답 언어: 항상 한국어로 답변
    - 길이: 핵심 먼저, 필요 시 상세 설명 추가
    - 형식: 가능하면 번호 목록이나 표 활용
    - 어조: 전문적이되 친근하게
    - 이모지: 사용하지 않음
    - 확인 질문: 지시 전에 명확히 이해했는지 확인 불필요, 바로 실행
    
    
    
    [개발자 특화 설정]
    - 코드 답변 시 항상 타입 힌트 포함
    - 에러 해결 시 원인 → 해결책 순서로 설명
    - 코드 블록에 언어 태그 항상 표기
    - 보안 취약점이 있을 경우 반드시 경고
    
    
    
    [마케터 특화 설정]
    - 마케팅 카피는 3개 버전으로 제안
    - 데이터/수치 인용 시 출처 표기
    - 글 작성 시 SEO 관점 피드백 포함
    - 경쟁사 언급 시 중립적 표현 사용
    

## 

## 개인정보 관리와 프라이버시 주의사항

Memory와 Custom Instructions에는 편의성만큼이나 프라이버시 위험도 존재합니다. 다음 사항을 반드시 확인하세요.

**주의해야 할 입력 정보**

  * 주민등록번호, 여권번호 등 신원 정보
  * 계좌번호, 카드 번호 등 금융 정보
  * 타인의 개인정보
  * 회사 기밀 프로젝트의 구체적인 수치나 계획


**Memory 사용 시 권장 사항**

  1. 주기적으로 저장된 Memory를 검토하고 불필요한 항목 삭제
  2. 공용 컴퓨터나 공유 계정에서는 민감한 정보 입력 금지
  3. 업무 종료 시 민감 정보가 담긴 Memory 삭제 습관화


tip

Temporary Chat 기능을 사용하면 Memory에 저장되지 않는 일회성 대화를 진행할 수 있습니다. 민감한 업무를 처리할 때는 새 채팅 창의 우측 상단 전구 아이콘을 클릭하여 Temporary Chat을 활성화하세요.

## 

## 실습: 나만의 업무 페르소나 설정하기

### 실습 시나리오

**목표** : Custom Instructions를 최적화하여 ChatGPT가 나의 업무 맥락을 자동으로 이해하도록 설정

**준비물** : ChatGPT 계정 (무료 포함)

### 단계 1: Custom Instructions 접근

우측 상단 이름/프로필 아이콘 클릭 → **"Customize ChatGPT"** 선택

### 단계 2: 나에 대한 정보 입력

자신의 직무에 맞는 내용으로 수정하여 첫 번째 섹션에 입력합니다.
    
    
    나는 IT 서비스 기업의 프로덕트 매니저(PM)입니다.
    - 담당 제품: B2C 모바일 앱 (MAU 50만)
    - 협업 대상: 개발팀, 디자인팀, 마케팅팀
    - 주요 업무: 요구사항 정의, 로드맵 수립, 데이터 분석
    - 사용 도구: Jira, Confluence, Figma, Amplitude
    - 현재 목표: 사용자 리텐션 개선 (현재 D30 리텐션 22% → 30% 목표)
    

### 단계 3: 응답 방식 설정

두 번째 섹션에 응답 형식을 입력합니다.
    
    
    - 항상 한국어로 답변해주세요
    - 실행 가능한 조언을 우선으로 제공해주세요
    - 불확실한 내용은 "확인 필요"로 표시해주세요
    - 표나 리스트를 적극 활용해주세요
    - 프레임워크 제안 시 장단점을 함께 설명해주세요
    - 답변이 길 경우 먼저 TL;DR 요약을 제공해주세요
    

### 단계 4: Memory에 프로젝트 정보 저장

ChatGPT와 대화를 시작하고 다음과 같이 입력합니다.
    
    
    다음 정보를 기억해줘:
    - 현재 Q2 OKR: 리텐션 D30 22%→30% 달성
    - 진행 중인 프로젝트: 온보딩 플로우 개편 (5월 출시 예정)
    - 팀 현황: 개발자 4명, 디자이너 1명
    - 주요 경쟁사: A앱, B앱
    

### 단계 5: 효과 확인

설정 완료 후 새 대화에서 아래 질문을 해보세요. Custom Instructions와 Memory가 자동으로 적용된 맞춤형 답변을 받을 수 있습니다.
    
    
    리텐션 개선을 위한 온보딩 개선 전략을 제안해줘.
    

tip

Custom Instructions는 언제든지 수정할 수 있습니다. 새로운 프로젝트가 시작되거나 직무가 바뀌면 즉시 업데이트하세요. 또한 Custom Instructions의 "Enable for new chats" 토글을 꺼두면 특정 대화에서는 적용을 일시 중단할 수 있습니다.
