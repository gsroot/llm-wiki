## 핵심 요약 (자가 작문)

10장은 ChatGPT를 범용 도구에서 "나만의 어시스턴트"로 전환하는 두 축, Memory와 Custom Instructions를 다룬다. Memory는 대화 중 자동으로 사용자 직무·선호·기술 스택 등을 누적 저장해 매번 맥락을 다시 입력할 필요를 없애 주고, Custom Instructions는 정체성과 응답 스타일을 사전에 명시하는 시스템 프롬프트 역할을 한다. 두 기능을 결합하면 동일한 사용자에게 일관된 톤과 도메인 전문성으로 응답하는 페르소나가 완성된다. 다만 민감 정보의 자동 저장 가능성을 인지하고, 주기적으로 저장 항목을 점검·삭제해 프라이버시를 관리하는 운영 습관이 필수다.
