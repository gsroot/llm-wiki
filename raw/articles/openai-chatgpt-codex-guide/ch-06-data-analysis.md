## Chapter 06. 데이터 분석에 활용하기 — Advanced Data Analysis

엑셀 파일을 열고, 수식을 짜고, 차트를 만들고, 인사이트를 뽑아내는 일련의 데이터 분석 작업은 전문적인 지식 없이는 쉽지 않습니다. ChatGPT의 **Advanced Data Analysis** 기능(구 Code Interpreter)은 이 모든 과정을 대화 한 번으로 처리해줍니다. 파일을 업로드하고 "이 데이터에서 월별 트렌드를 분석해줘"라고 말하는 것만으로 Python 코드 작성, 분석 실행, 그래프 생성까지 자동으로 수행됩니다.

* * *

## 6.1 Advanced Data Analysis란?

Advanced Data Analysis는 ChatGPT 내부에서 **실제 Python 코드를 실행**하여 데이터를 분석하는 기능입니다. 사용자가 Python을 전혀 몰라도 자연어로 요청하면 ChatGPT가 코드를 작성하고 실행한 뒤 결과를 보여줍니다.

### 주요 특징

특징 | 설명  
---|---  
코드 자동 작성 | 분석 요청을 Python 코드로 변환해 실행  
파일 업로드 지원 | CSV, Excel, JSON, PDF 등 다양한 형식 처리  
시각화 자동 생성 | 막대 그래프, 선 그래프, 히트맵 등 이미지로 출력  
반복 수정 가능 | "색상 바꿔줘", "X축 레이블 추가해줘"처럼 대화로 수정  
  
### 활성화 방법

  1. ChatGPT Plus 또는 Teams/Enterprise 구독 필요
  2. 대화 시작 시 모델을 **GPT-4o** 로 선택
  3. 파일 업로드 아이콘(클립)을 클릭해 파일 첨부
  4. 자연어로 분석 요청

tip

Advanced Data Analysis는 ChatGPT Plus($20/월) 이상 구독자에게 제공됩니다. 무료 사용자는 텍스트로 데이터를 붙여넣고 분석을 요청하는 방식으로 일부 기능을 활용할 수 있습니다.

## 6.2 파일 업로드 방법과 지원 형식

ChatGPT에 데이터 파일을 업로드하는 방법은 간단합니다.

### 업로드 절차

  1. 대화창 하단의 **클립(📎) 아이콘** 클릭
  2. 로컬 파일 탐색기에서 분석할 파일 선택
  3. 업로드 완료 메시지 확인 후 분석 요청 입력


### 지원 파일 형식

형식 | 확장자 | 주요 용도  
---|---|---  
엑셀 | .xlsx, .xls | 매출 데이터, 설문 결과  
CSV | .csv | 대용량 데이터셋, 로그 데이터  
JSON | .json | API 응답 데이터, 설정 파일  
  
### 업로드 시 주의사항

  * 파일 크기는 일반적으로 **512MB 이하** 권장
  * 개인정보(이름, 주민번호, 연락처)가 포함된 파일은 업로드 전 마스킹 처리 필수
  * 첫 번째 행이 **헤더(열 이름)** 인 형식이 가장 분석하기 좋음
  * 한글 열 이름도 인식하지만, 영문 열 이름이 더 안정적으로 동작하는 경우 있음

tip

업로드 후 먼저 "이 파일의 구조를 설명해줘"라고 요청해보세요. ChatGPT가 열 이름, 데이터 타입, 행 수, 기본 통계를 정리해줘서 어떤 분석이 가능한지 파악하기 쉽습니다.

## 6.3 데이터 분석 워크플로

Advanced Data Analysis를 효과적으로 활용하려면 아래 5단계 흐름을 따르는 것이 좋습니다.

### 단계별 분석 흐름

**1단계 — 업로드(Upload)**
    
    
    파일을 업로드한 후:
    "이 파일의 전체 구조와 기본 정보를 알려줘.
    열 이름, 데이터 타입, 총 행 수, 결측값 여부를 포함해서 요약해줘."
    

**2단계 — 탐색(Explore)**
    
    
    "각 열의 기술 통계(평균, 중앙값, 최솟값, 최댓값, 표준편차)를
    표 형식으로 정리해줘."
    

**3단계 — 분석(Analyze)**
    
    
    "월별 매출 합계를 계산하고, 전월 대비 증감률도 함께 보여줘."
    

**4단계 — 시각화(Visualize)**
    
    
    "위 월별 매출 데이터를 선 그래프로 그려줘.
    X축은 월, Y축은 매출액(만원 단위), 제목은 '2025년 월별 매출 추이'로 설정해줘."
    

**5단계 — 인사이트(Insight)**
    
    
    "이 데이터를 바탕으로 주목할 만한 패턴이나 이상치가 있다면 알려줘.
    비즈니스 관점에서 3가지 인사이트를 도출해줘."

## 6.4 자주 쓰는 분석 유형

### 기술 통계 (평균, 중앙값, 표준편차)

데이터의 전반적인 분포를 파악하는 가장 기본적인 분석입니다.
    
    
    업로드한 파일에서 '매출액' 열의 기술 통계를 계산해줘.
    평균, 중앙값, 최솟값, 최댓값, 표준편차를 포함하고,
    이 값들이 의미하는 바를 쉽게 설명해줘.
    

### 피벗 테이블 생성

여러 기준으로 데이터를 집계할 때 유용합니다.
    
    
    '지역'과 '제품명'을 기준으로 매출 합계 피벗 테이블을 만들어줘.
    행은 지역, 열은 제품명, 값은 매출액 합계로 설정해줘.
    

### 트렌드 분석 (시계열)

시간에 따른 변화를 파악하는 분석입니다.
    
    
    '날짜' 열을 기준으로 월별 매출 합계를 집계하고,
    3개월 이동평균(Rolling Mean)도 함께 계산해서 선 그래프로 그려줘.
    

### 이상치 탐지

예상 범위를 벗어난 데이터를 찾아냅니다.
    
    
    '매출액' 열에서 IQR(사분위 범위) 방법으로 이상치를 탐지해줘.
    이상치로 판단된 행을 별도 표로 보여주고,
    박스 플롯(Box Plot)도 그려줘.
    

### 상관관계 분석

두 변수 사이의 관계 강도를 파악합니다.
    
    
    '광고비', '매출액', '방문자수' 열 간의 상관계수를 계산하고,
    상관관계 히트맵으로 시각화해줘.
    어떤 변수가 매출에 가장 큰 영향을 미치는지 해석해줘.

## 6.5 시각화 유형 가이드

ChatGPT는 다양한 그래프 유형을 지원합니다. 데이터 특성에 따라 적합한 시각화를 선택하세요.

그래프 유형 | 적합한 상황 | 프롬프트 키워드  
---|---|---  
막대 그래프 | 항목 간 비교 | "막대 그래프(bar chart)"  
선 그래프 | 시간에 따른 변화 | "선 그래프(line chart)"  
산점도 | 두 변수의 관계 | "산점도(scatter plot)"  
히트맵 | 상관관계, 빈도 행렬 | "히트맵(heatmap)"  
  
### 시각화 품질 높이는 프롬프트
    
    
    선 그래프를 그릴 때 다음 옵션을 적용해줘:
    - 제목: '2025년 월별 매출 및 3개월 이동평균'
    - X축 레이블: '월'
    - Y축 레이블: '매출액 (만원)'
    - 매출 선: 파란색 실선
    - 이동평균 선: 주황색 점선
    - 범례 표시
    - 그리드 라인 추가
    - 해상도: 150 DPI
    

tip

그래프를 생성한 후 "이 그래프를 PNG 파일로 다운로드할 수 있게 해줘"라고 요청하면 다운로드 링크를 제공합니다. 보고서에 바로 삽입할 수 있는 고해상도 이미지를 얻을 수 있습니다.

## 6.6 실습: 매출 데이터 CSV로 월별 트렌드 분석 + 시각화 보고서 만들기

이번 실습에서는 가상의 매출 데이터 CSV 파일을 업로드하고, 단계별 프롬프트를 통해 분석 보고서를 완성하는 전체 과정을 따라합니다.

### 실습 시나리오

**상황** : 이커머스 업체 운영팀의 박지수 씨는 2025년 연간 매출 데이터를 분석해야 합니다. 엑셀은 다룰 줄 알지만 고급 분석은 어렵습니다. ChatGPT Advanced Data Analysis로 월별 트렌드 분석과 상위 5개 제품 비교 보고서를 만들려고 합니다.

### 실습용 샘플 CSV 구조

실습을 위해 아래와 같은 구조의 CSV 파일을 사용합니다.
    
    
    날짜,제품명,카테고리,매출액,판매수량,지역
    2025-01-05,스마트워치 A,전자기기,850000,1,서울
    2025-01-12,운동화 B,패션,120000,2,부산
    2025-01-18,에어팟 C,전자기기,320000,1,서울
    2025-02-03,스마트워치 A,전자기기,850000,1,경기
    ...
    

### 단계별 프롬프트 가이드

**Step 1 — 데이터 구조 파악**
    
    
    방금 업로드한 CSV 파일의 구조를 분석해줘.
    - 총 행 수와 열 수
    - 각 열의 데이터 타입
    - 날짜 범위
    - 결측값 여부
    를 표로 정리해줘.
    

**Step 2 — 월별 매출 집계**
    
    
    '날짜' 열을 월 단위로 집계해서 월별 총 매출액을 계산해줘.
    전월 대비 증감액과 증감률(%)도 함께 계산하고,
    표로 정리해줘.
    

**Step 3 — 월별 트렌드 시각화**
    
    
    월별 매출액 추이를 선 그래프로 그려줘.
    - 제목: '2025년 월별 매출 트렌드'
    - X축: 월 (1월~12월)
    - Y축: 매출액 (만원)
    - 매출이 가장 높은 달과 가장 낮은 달에 마커와 라벨 표시
    - 전체적인 추세선(trend line)도 함께 표시
    

**Step 4 — 상위 5개 제품 분석**
    
    
    이 엑셀 파일의 월별 매출 추이를 분석하고,
    상위 5개 제품의 매출 비교 막대 그래프를 그려줘.
    
    추가로:
    - 각 제품의 연간 매출 합계와 비중(%)을 표로 보여줘
    - 상위 5개 제품이 전체 매출에서 차지하는 비율도 계산해줘
    

**Step 5 — 종합 인사이트 도출**
    
    
    지금까지 분석한 내용을 바탕으로 경영진에게 보고할 수 있는
    인사이트 요약을 작성해줘.
    
    형식:
    1) 2025년 매출 하이라이트 (3줄)
    2) 주목할 패턴 또는 이상치 (2~3가지)
    3) 다음 분기 전략 제언 (2~3가지)
    
    비전문가도 이해할 수 있도록 쉬운 언어로 작성해줘.
    

### 실습 결과 예시

단계를 모두 수행하면 다음과 같은 결과물을 얻게 됩니다.
    
    
    [Step 2 결과 예시]
    
    | 월 | 매출액 | 전월 대비 증감 | 증감률 |
    |----|--------|--------------|--------|
    | 1월 | 12,450만원 | - | - |
    | 2월 | 10,230만원 | -2,220만원 | -17.8% |
    | 3월 | 14,890만원 | +4,660만원 | +45.6% |
    ...
    | 11월 | 28,340만원 | +5,100만원 | +21.9% |
    | 12월 | 35,720만원 | +7,380만원 | +26.0% |
    

tip

분석 중간에 "이 부분 더 자세히 알려줘"나 "다른 방식으로 시각화해봐"처럼 대화를 이어가면 됩니다. ChatGPT는 이전 분석 결과를 기억하고 있어서 연속적인 심층 분석이 가능합니다. 분석이 완료되면 "지금까지의 분석 내용을 보고서 형식으로 정리해줘"라고 요청해 전체 내용을 하나의 문서로 만들 수도 있습니다.

## 6.7 분석 결과를 보고서로 정리하기

분석이 끝났다면 모든 내용을 하나의 보고서로 통합할 수 있습니다.
    
    
    지금까지 수행한 분석 결과를 바탕으로
    경영진 보고용 데이터 분석 보고서를 작성해줘.
    
    형식:
    1. 분석 개요 (목적, 데이터 출처, 분석 기간)
    2. 주요 분석 결과 (월별 트렌드, 상위 제품 현황)
    3. 핵심 인사이트 3가지
    4. 전략적 제언
    5. 다음 단계 (추가로 분석이 필요한 부분)
    
    전문적이고 간결하게 작성하되, 비데이터 전문가도 이해할 수 있게 써줘.
    

이처럼 ChatGPT Advanced Data Analysis는 데이터 전문가가 아니어도 수준 높은 데이터 분석과 시각화를 수행할 수 있게 해줍니다. 엑셀에서 몇 시간이 걸리던 작업을 단 몇 분 만에 완성할 수 있습니다.
