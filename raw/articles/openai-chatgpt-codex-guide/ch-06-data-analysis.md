## 핵심 요약 (자가 작문)

이 장은 ChatGPT의 Advanced Data Analysis(과거 Code Interpreter) 기능을 활용해 비전문가도 데이터 분석을 자연어로 수행하는 방법을 다룬다. CSV·Excel·JSON 등 다양한 파일을 업로드한 뒤 "월별 트렌드를 분석해줘" 같은 일상 한국어로 요청하면 ChatGPT가 내부적으로 Python 코드를 작성·실행해 통계 계산과 그래프 생성까지 자동으로 수행한다. 저자는 업로드 → 데이터 탐색 → 분석 → 시각화 → 인사이트 추출의 5단계 표준 워크플로를 제시하고, 자주 쓰는 분석 유형과 시각화 차트 가이드를 표로 정리한다. 매출 CSV로 월별 트렌드를 분석하고 시각화 보고서까지 만드는 실습은, 엑셀로 몇 시간 걸리던 작업을 몇 분 만에 끝내는 가능성을 직접 체험하게 한다.

## 인용 (key quotes, 짧게)
