## 핵심 요약 (자가 작문)

13장은 Codex를 처음 접하는 개발자에게 정체성·도구 비교·환경 구성을 차례로 안내한다. Codex는 채팅창에 자연어로 요청하면 GitHub 리포지토리를 탐색해 코드를 작성하고 PR을 직접 생성하는 클라우드 자율 에이전트다. GitHub Copilot이 IDE 인라인 자동완성에 강하고 Claude Code가 로컬 터미널 페어 프로그래밍에 강한 반면, Codex는 비동기 백그라운드 태스크 실행에 적합해 세 도구는 대체재가 아닌 보완재로 정리된다. 환경 설정 단계에서는 GitHub 연동, AGENTS.md(에이전트가 따라야 할 코딩 컨벤션·테스트 명령 정의), Setup Script(샌드박스 부팅 시 의존성 설치)를 다루고, 첫 실습은 단순 API 엔드포인트를 추가하면서 PR이 생성되는 전체 흐름을 체험한다.

## 인용 (key quotes, 짧게)

> "세 도구는 서로 대체재가 아니라 보완재입니다."
