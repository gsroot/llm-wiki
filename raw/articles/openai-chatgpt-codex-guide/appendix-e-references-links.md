## E. 참고 자료 및 공식 문서 링크

ChatGPT와 OpenAI API를 더 깊이 학습하고 최신 정보를 유지하는 데 필요한 공식 문서, 커뮤니티, 학습 리소스를 모았습니다.

## OpenAI 공식 문서

리소스 | URL | 설명  
---|---|---  
ChatGPT 도움말 | https://help.openai.com | ChatGPT 사용법, 요금제, 계정 문의  
API 문서 | https://platform.openai.com/docs | API 레퍼런스, 가이드, 모델 정보  
OpenAI Cookbook | https://cookbook.openai.com | 실전 코드 예제 및 레시피 모음  
Playground | https://platform.openai.com/playground | API 파라미터 실험 및 프롬프트 테스트  
서비스 상태 | https://status.openai.com | 실시간 장애 현황 및 인시던트 이력  
요금 안내 | https://openai.com/pricing | 최신 API 및 ChatGPT 요금  
tip

API 개발 중 응답이 이상하게 느리거나 오류가 많을 때는 **https://status.openai.com** 을 먼저 확인하세요. 서버 장애인지 내 코드 문제인지 빠르게 구분할 수 있습니다.

## 커뮤니티

### OpenAI 공식 커뮤니티

  * **OpenAI Community Forum** : https://community.openai.com
  * API 사용 질문, 버그 리포트, 사용 사례 공유
  * OpenAI 직원들이 직접 답변하는 공식 채널


### Reddit

  * **r/ChatGPT** : https://reddit.com/r/ChatGPT
  * 일반 사용자들의 ChatGPT 활용 사례와 팁 공유
  * 신기능 발표 시 커뮤니티 반응을 가장 빠르게 확인 가능

  * **r/OpenAI** : https://reddit.com/r/OpenAI

  * OpenAI 관련 뉴스·연구·정책 토론

  * **r/LocalLLaMA** : https://reddit.com/r/LocalLLaMA

  * 로컬 AI 모델과 OpenAI API를 비교·병행하는 개발자 커뮤니티


### Discord

  * **OpenAI Discord** : https://discord.gg/openai
  * 실시간 Q&A, API 개발자 채널, 새 기능 알림

  * **AI Builders** : 다양한 AI 개발자 Discord 서버

  * "AI Builders" 검색으로 다수의 활성 서버 찾을 수 있음

### 온라인 강좌

**DeepLearning.AI — ChatGPT Prompt Engineering for Developers**

  * 주소: https://learn.deeplearning.ai/chatgpt-prompt-eng
  * 강사: Isa Fulford (OpenAI), Andrew Ng (DeepLearning.AI)
  * 내용: 프롬프트 엔지니어링 원칙, 요약·추론·변환·확장 실습
  * 수준: 개발자 입문~중급 / 무료 제공


**DeepLearning.AI — Building Systems with ChatGPT API**

  * 주소: https://learn.deeplearning.ai/building-systems-with-chatgpt
  * 내용: 멀티스텝 파이프라인, 평가 시스템 구축
  * 수준: 중급 / 무료 제공


**Prompt Engineering Guide**

  * 주소: https://promptingguide.ai
  * 내용: Chain-of-Thought, Few-Shot, ReAct 등 고급 프롬프트 기법 정리
  * 한국어 번역본 제공


### YouTube 채널
