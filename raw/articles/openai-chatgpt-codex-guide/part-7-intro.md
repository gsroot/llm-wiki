# Part 7. API 연동과 자동화

## 페이지 구조 (헤딩 outline)
- Part 7. API 연동과 자동화 (intro)

## 핵심 요약 (자가 작문)

이 파트는 웹 인터페이스로 ChatGPT를 쓰던 단계를 넘어, OpenAI API를 통해 자신의 애플리케이션에 GPT 모델을 직접 통합하는 방향으로 넘어간다. 이메일 자동 분류, 고객 상담 챗봇, 반복 업무 자동화 같은 실무형 워크플로를 코드 몇 줄로 구축할 수 있다는 점이 강조된다. API 키 발급과 환경 설정부터 Chat Completions, Assistants API, Zapier·Slack·Google Sheets 같은 자동화 도구 연동까지 단계적으로 다루며, 프로그래밍 경험이 부족한 독자도 따라갈 수 있도록 구성됐다. 목표는 단순 사용자에서 자신만의 AI 워크플로 설계자로 역량을 끌어올리는 것이다.

## 인용 (key quotes, 짧게)

> "코드 몇 줄로 이메일을 자동 분류하고, 고객 상담 봇을 만들고, 반복 업무를 자동화하는 워크플로를 구축할 수 있습니다."

> "AI를 단순히 '사용하는' 수준을 넘어, 나만의 AI 워크플로를 설계하고 구현할 수 있는 역량을 갖추게 됩니다."
