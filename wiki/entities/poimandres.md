# Poimandres

## 개요

**Poimandres** (`pmndrs`)는 다국적 React **OSS Collective**. 단일 회사가 아닌 분산된 메인테이너 그룹으로, [[zustand]] / Jotai / Valtio (3대 상태관리) + react-three-fiber + drei (Three.js React 통합) 등을 운영.

본 페이지는 **stub** — pmndrs 메타 저장소가 raw 수집 대상이 아닌 상태에서 22회차 [[zustand]] / [[pmndrs-zustand]] 페이지가 참조하므로 정합성 stub으로 등록.
