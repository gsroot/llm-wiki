# 포트폴리오 시드 문서 (old-portfolio.md)

## 한줄 요약

> 석근의 9년차 백엔드/풀스택 커리어 전체를 한 문서에 압축한 초기 포트폴리오. 컴투스플랫폼(Com2usPlatform) 애널리틱스(BI)·블록체인·ML 프로젝트와 줌인터넷 초기 경력, 최근 개인 프로젝트(트래블메이트/Mate Chat/카카오톡 분석)까지 망라한다.

## 핵심 내용

- **자기 소개 & 연락처** — Python 기반 백엔드·데이터 파이프라인·ML·BI를 두루 경험한 9년차 개발자. GitHub @gsroot, Blog gsroot.tistory.com, LinkedIn.
- **기술 스택** — Python(FastAPI/Django/Flask/Spring Boot/NestJS), React/Mobx, GCP BigQuery·AutoML·AI Platform Pipeline, Docker/Jenkins/GitHub Actions, MySQL/PostgreSQL/Redis, Flutter, Rust(스마트 컨트랙트), LangGraph/LangChain/OpenAI API.
- **학력·자격** — 인하대 컴공(2007–2014), 패스트캠퍼스 파이썬 데이터 분석(2016), ADSP·정보처리기사, MSP 수상(Imagine Cup).
- **경력 프로젝트** — 시간 역순으로 10개 이상:
  - 컴투스플랫폼: 애널리틱스 React 리뉴얼(2025.06–) · 에어브릿지 API(2025.01–02) · 공통 모듈 개선(2024.08–12) · XPLA 플랫폼(2024.04–07) · NFT 마켓(2022.05–2024.03) · CODE 트래블룰(2021.10–2022.04) · 통합 인증 모듈(2021.06–07) · ML 유저 예측(2020.08–2021.09) · 대용량 다운로드 API(2019.03–06) · 스케줄러(2018.07–08) · 애널리틱스 본체(2017.05–).
  - 줌인터넷: 스윙 브라우저 플러그인(2016.01–07), 줌닷컴 데이터 분석(2016.02–06).
- **개인 프로젝트** — Mate Chat(AI 소셜 메시징, 2025.08–), 트래블메이트(Google Play 출시, 2025.10–11), 카카오톡 대화 분석(2023.01–08).

## 주요 인사이트

- 이 문서가 이후 `docs/50-portfolio/*`, `docs/20-projects/*` 체계의 **시드**가 됨. 내용은 Johnny.Decimal + 3-Layer 구조로 재가공됨.
- 성과 서술 패턴: *프로젝트 요약 → 역할 → 기술 스택 → 성과* 4블록이 거의 모든 프로젝트에 공통.
- 정량 지표가 후속 통합 문서에서 강화됨: 가스비 90% 절감(NFT 마켓), ML 예측 정확도 85%+, 공통 API 단독 유지보수 92%, 프론트엔드 생산성 30~40% 향상 기반.
